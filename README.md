# blockaft
一个关于竞争与合作的游戏。

网页上面的游戏，考虑用[Egret](https://www.egret.com), TypeScript开发

游戏设计思路请见design.md

## 运行
使用Egret Wing打开项目目录（web-src目录），点击虫子按钮debug

或者，可以在命令行，进入到web-src目录，运行
~~~
egret run
~~~
更多命令，输入egret参考
